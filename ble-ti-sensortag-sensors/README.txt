1) Download evothings studio https://evothings.com/download/
2) Follow instructions on website and obtain cloud token
3) Once evothings viewer and studio is installed, connect to the Studio using "Get key" under the "Connect" tab
4) Go to the "Build" tab. Drag and drop the file "ble-ti-sensortag-sensors" into the window.
5) Click on run
6) Your phone should start running the application (no cable needed, fully wireless)
